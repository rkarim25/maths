# Curriculum Documentation

This document outlines the curriculum structure for Liyana's Maths Adventure, aligned with UK National Curriculum and 11+ exam preparation.

## Overview

The curriculum is structured across 4 years:
1. **Year 1 (Foundation Year)** - Ages 6.5-7.5
2. **Year 2 (Explorer Year)** - Ages 7.5-8.5
3. **Year 3 (Challenger Year)** - Ages 8.5-9.5
4. **Year 4 (11+ Sprint Year)** - Ages 9.5-10.5

Each year is divided into 3 terms:
- Autumn Term
- Spring Term
- Summer Term

Each term contains approximately 6-8 episodes, with each episode consisting of:
- A story-slide lesson (teaching)
- A story-game practice (application)
- A mini-assessment (mastery check)

## Progression Logic

### Mastery Threshold
Each episode requires an 80% score on the mini-assessment to be considered "completed" and unlock the next episode.

### Unlocking Episodes
Episodes unlock sequentially within each strand. Mastery of all episodes in a term is recommended before progressing to the next term.

### Weak Area Detection
The system tracks accuracy per skill tag. If accuracy falls below:
- 50%: High severity weak area
- 50-69%: Medium severity weak area
- 70-79%: Low severity weak area
- 80%+: Skill is considered mastered

### Remediation
When a high severity weak area is detected, the system recommends remedial episodes targeting that specific skill.

## Assessment Structure

### Mini-Assessments
Each episode concludes with a 5-question mini-assessment:
- Question types: Multiple choice, drag-and-drop, input
- Time limit: None for Year 1-2, 5 minutes for Year 3-4
- Pass threshold: 80% (4 out of 5 questions)

### Term Reviews
Each term concludes with a comprehensive review episode covering all skills from that term.

### Year Reviews
Each year concludes with a comprehensive review episode covering all skills from that year.

### Mock Exams (Year 4)
Year 4 includes 4 full mock exams:
- 50 questions each
- 50-minute time limit
- Mixed question types
- Detailed performance analysis
